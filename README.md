# 📊 Superstore Sales Analysis & Dashboard

## 🚀 Project Overview

This project focuses on analyzing a retail Superstore dataset to uncover key sales trends and business insights. The analysis is performed using Python (Pandas) and visualized through both static charts and an interactive dashboard built with Streamlit.

The goal is to transform raw data into actionable insights that can support data-driven decision-making.

---

## 🎯 Objective

To analyze the Superstore dataset and identify patterns in sales across categories, sub-categories, regions, customer segments, and time. The project aims to highlight key revenue drivers, top-performing products, and seasonal trends.

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas 📊
* Matplotlib 📈
* Streamlit 🌐

---

## 📂 Dataset

* Superstore Sales Dataset (CSV format)
* Contains information about orders, customers, products, regions, and sales

---

## 🧹 Data Cleaning Steps

* Cleaned column names for consistency
* Converted date columns (`order_date`, `ship_date`) to datetime format
* Handled missing values in postal codes and dates
* Removed duplicate records
* Created new feature: `order_month` for time-based analysis

## 📊 Key Insights

### 📌 Category Analysis

* Technology category generates the highest revenue
* Indicates strong demand for high-value tech products

### 📌 Sub-Category Analysis

* Phones lead in sales among all sub-categories
* High-demand essential products dominate revenue

### 📌 Regional Analysis

* West region is the top-performing region
* Highlights strong market presence compared to other regions

### 📌 Product Analysis

* Canon imageCLASS 2200 Advanced Copier is the top-selling product
* Premium products significantly contribute to total revenue

### 📌 Customer Segment Analysis

* Consumer segment contributes the highest sales
* Business relies heavily on individual customers

### 📌 Monthly Trend

* February shows higher sales compared to several other months
* Indicates possible seasonal trends in customer purchasing behavior

## 🌐 Streamlit Dashboard

An interactive dashboard is built using Streamlit with:

* Region-based filtering
* Dynamic charts
* Real-time data exploration

---

## 💡 Conclusion

The analysis reveals that revenue is primarily driven by the Technology category, high-value products, and the Consumer segment. The West region dominates sales, while monthly trends suggest seasonal variations. Focusing on high-performing areas and improving underperforming segments can help drive business growth.
